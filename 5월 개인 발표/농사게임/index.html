    기본 조작:

    • 도구 선택:
      • 씨앗: 🛒 아이콘을 눌러 상점을 여세요. 원하는 씨앗을 구매한 후, 화면 상단의 해당 씨앗 버튼 (예: 🥕)을 클릭하여 씨앗 도구를 선택합니다.
      • 요리: 🧑‍🍳 아이콘을 누르면 부엌으로 이동할 수 있습니다. 키우신 작물을 요리해서 더 높은 가격에 팔아보세요.
      • 수확 도구: 낫(🌾) 모양의 수확 도구 버튼을 클릭하여 선택합니다.
    • 씨앗 심기: 씨앗 도구를 선택한 상태에서 비어있는 밭을 클릭하면 씨앗이 심어집니다. 씨앗은 시간이 지나면 자동으로 자랍니다.
    • 수확하기: 수확 도구를 선택한 후, 다 자란 작물(아이콘이 바뀜)을 클릭하여 수확합니다. 수확한 작물은 왼쪽 창고로 이동합니다.
    • 작물 판매하기: 창고에 있는 작물을 클릭하면 즉시 판매되어 돈을 얻을 수 있습니다.

    화면 상단에서 현재 가진 돈과 선택된 도구를 확인할 수 있습니다.
